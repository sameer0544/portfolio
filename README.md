# Sameer Borkar - Portfolio 🏎️

Personal portfolio of Sameer Borkar, an Engineer, Developer, and Designer with 7+ years of experience in IoT, Embedded Systems, and Web Development.

## 🌟 Live Demo
Check out the live site: [https://sameer0544.github.io/portfolio/](https://sameer0544.github.io/portfolio/)

## 🚀 Features
- **Drifting Ferrari Cursor**: A physics-based Ferrari SVG that follows your cursor and drifts on sharp turns.
- **Light Garage Theme**: A premium, industrial workshop aesthetic tailored to hardware and automotive engineering.
- **Interactive Project Modals**: Click on any project to view a detailed report, tech stack, and key features.
- **Responsive Design**: Fully functional on desktop and mobile.
- **Resume Download**: Integrated 1-click PDF resume download.

## 🛠️ Tech Stack
- **HTML5 & CSS3**: Vanilla CSS with glassmorphism and modern typography.
- **JavaScript**: Physics engine for the car cursor and dynamic modal handling.
- **GitHub Pages**: Automated deployment and hosting.

## 📁 Project Structure
- `index.html`: Main site structure and content.
- `styles.css`: Complete styling for the "Light Garage" theme.
- `script.js`: Mouse-following car physics and interactive UI logic.
- `images/`: High-quality project photographs and assets.
- `resume.pdf`: Professional resume with 7+ years of experience.
